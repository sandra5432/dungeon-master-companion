# Features — A: Marktplatz

| Feature ID | Feature Name | Description | Access |
|------------|--------------|-------------|--------|
| AL-A-001 | Artikelliste anzeigen | Alle Marktplatz-Artikel werden in einer tabellarischen Übersicht mit Name, Preis, Attribut, Tags und Link angezeigt. | Alle (auch nicht eingeloggt) |
| AL-A-002 | Artikel suchen | Freitextsuche filtert die angezeigte Artikelliste nach Name in Echtzeit. | Alle (auch nicht eingeloggt) |
| AL-A-003 | Artikel nach Preis filtern | Zwei Eingabefelder (Min. Preis / Max. Preis) schränken die angezeigte Artikelliste auf eine Preisspanne ein. | Alle (auch nicht eingeloggt) |
| AL-A-004 | Artikel nach Tag filtern | Ein Dropdown mit Checkboxen filtert die Artikelliste auf Artikel mit einem oder mehreren ausgewählten Tags. | Alle (auch nicht eingeloggt) |
| AL-A-005 | Artikelliste sortieren | Klick auf den Tabellenkopf „Name" oder „Preis ⚜" sortiert die Liste auf- oder absteigend nach dem jeweiligen Feld. Ein zweiter Klick kehrt die Reihenfolge um. | Alle (auch nicht eingeloggt) |
| AL-A-006 | Artikel erstellen | Über ein Modal können Name, Preis, Attribut, Tags, Link und Beschreibung eines neuen Artikels erfasst und gespeichert werden. | Admin only |
| AL-A-007 | Artikel bearbeiten | Bestehende Artikel können über ein Modal aktualisiert werden (alle Felder editierbar). | Admin only |
| AL-A-008 | Artikel löschen | Ein Artikel wird nach Bestätigung dauerhaft aus der Datenbank entfernt. | Admin only |
| AL-A-009 | Artikelliste als Markdown exportieren | Alle Artikel der Datenbank (unabhängig von aktiven Filtern) werden als `.md`-Datei mit Markdown-Tabelle heruntergeladen. | Admin only |
