# Features — Allgemein

| Feature ID | Feature Name | Description                                                                                                                                                                                  | Access |
|------------|--------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| AL-G-001 | Einloggen | Nutzer können sich mit Benutzername und Passwort einloggen. Nach erfolgreichem Login werden rollenabhängige UI-Elemente (Admin-Funktionen, Welt-Aktionen) eingeblendet.                      | Alle |
| AL-G-002 | Ausloggen | Ein eingeloggter Nutzer kann sich über einen Button abmelden. Die Session wird serverseitig beendet und die Ansicht wechselt in den nicht-eingeloggten Zustand auf den Marktplatz.           | Eingeloggte Nutzer |
| AL-G-003 | Dark / Light Mode umschalten | Ein Button in der Navigationsleiste wechselt zwischen Dark Mode (🌙) und Light Mode (☀️). Die Auswahl wird im Browser (localStorage) gespeichert und beim nächsten Besuch wiederhergestellt. | Alle |
